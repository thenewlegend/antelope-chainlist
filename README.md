# Antelope-Chainlist

### One-Click Connect Antelope EVM chains to Metamask etc.

A comprehensive list of Antelope Ethereum Virtual Machine (Antelope EVM) networks has been compiled for user reference. With this information, individuals can easily connect their wallets  to the appropriate Chain ID thereby ensuring seamless connectivity to the correct Antelope EVM networks, both mainnets and testnets.( Refer [Chainlist](https://chainlist.org/) )
